# WordPress Portfolio Website

This is a personal portfolio website developed using WordPress to demonstrate web development skills and hands-on practice with real-world website creation.

## Features
- Custom WordPress theme structure
- Responsive layout (mobile and desktop support)
- Portfolio section to showcase projects
- Contact form integration using plugins
- SEO friendly pages
- Optimized images and performance

## Technologies Used
- WordPress CMS  
- HTML5  
- CSS3  
- JavaScript (basic)  
- PHP (basic)  
- MySQL  
- Git & GitHub  

## Responsibilities Covered
- Website development and maintenance  
- Theme customization  
- Plugin usage and configuration  
- Responsive design implementation  
- Basic testing and bug fixing  
- Version control using Git  

## Setup Instructions
1. Install XAMPP  
2. Copy project folder to htdocs  
3. Create database in phpMyAdmin  
4. Configure wp-config.php  
5. Run the project on localhost
